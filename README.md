# DPAPROJECTSIP
Proyek ini merupakan aplikasi kasir sederhana berbasis Python yang digunakan untuk pengolahan data barang dan transaksi penjualan dengan penyimpanan data menggunakan file teks (.txt)

FITUR YANG DIIMPLEMENTASIKAN
Pengolahan data barang (tambah, hapus, edit)
Menampilkan data barang
Transaksi kasir berdasarkan kode barang
Pencatatan history transaksi
Penyimpanan data otomatis ke file .txt
